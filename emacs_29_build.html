Mac に Emacs 29.1 をインストールする

はじめに

MacBook Pro (Apple M2, Ventura 13.5, 2022) に GNU Emacs をインストールしました。インストールオプションとしてネイティブコンパイル、tree-sitter、xwidgets を導入しました。

準備

Emacs 29.1 のネイティブコンパイル機能と tree-sitter を使うために Homebrew で関連ツールをインストールしました。 

brew install gcc
brew install libgccjit
brew install tree-sitter

Emacs のインストール

まず GNU Emacs の日本のミラーサイト（http://ftp.jaist.ac.jp/pub/GNU/emacs/）から emacs-29.1.tar.gz をダウンロード、展開します。展開したファイルは ~/Downloads/ にあるとします。 

cd ~/Downloads/emacs-29.1/

でソースファイルに移動して 

./configure --help

でインストールオプションを確認します。今回は native compilation, tree-sitter, xwidgets 機能を導入します。

次の4つのコマンドを実行すると ~/Download/emacs-29.1/nextstep/ に Emacs.app が作成されました。 

./autogen.sh
./configure --with-xwidgets --with-native-compilation --with-tree-sitter
make clean
make install

Emacs.app を ~/Applications に移動させます。そのまま Emacs を開こうとしても "Emacs.app is damaged and can't be opend." というメッセージが表示されます。次のコマンドを実行するとうまく開けるようになりました。 

xattr -d com.apple.quarantine /Applications/Emacs.app
