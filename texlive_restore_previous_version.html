TeX Live のパッケージの復元

TeX Live のパッケージを tlmgr でアップデートすると nicematrix に不具合がありコンパイルがうまくできなくなりました。このパッケージを更新前のものに戻します。

パッケージを復元する

ターミナルで 

tlmgr info nicematrix

と入力。出力結果の一部に revision: 64433 とリビジョン番号が表示されました。 tlmgr restore [パッケージ名] でコンピュータ上にバックアップされているパッケージのすべてのリビジョンが表示されます。今回は 

tlmgr restore nicematrix

と入力して 

Available backups for nicematrix: 64185 (2022-09-20 07:03)

と表示されました。 sudo tlmgr restore [パッケージ名] [リビジョン番号] で指定したリビジョンを復元します。 

sudo tlmgr restore nicematrix 64185

これでうまくコンパイルできるようになりました。
